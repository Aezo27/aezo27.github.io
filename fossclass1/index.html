Pengenalan PHP
  Apa itu PHP, PHP merupakan akronim dari Hypertext Preprocessor adalah bahasa pemograman yang didesain khusus untuk pengembangan web, PHP bekerja pada server side karena dieksekusi dari server bukan dari komputer client seperti html maupun css, untuk selengkapnya bisa di cek di https://www.php.net.

Syntax PHP
   Pada PHP penulisannya dimulai dengan tanda (<?php) dan diakhiri dengan (?>), pada PHP versi 5, penulisannya bisa memakai ( <?) dan daiakhiri dengan (?>). Namun saat ini PHP 5 telah ditinggalkan dan berganti menjadi PHP 7.

<?php
echo "Fossclass FOSTI UMS";
?>

Dari contoh diatas PHP dimulai dengan tag pembuka (<?php) dan ditutup (?>) dan untuk echo sendiri digunakan untuk mencetak atau menampilkan sesuatu.

Installasi Server
  Server sendiri digunakan untuk mengakses PHP, seperti yang sudah saya sebelumnya PHP bekerja pada sisi server, apa bila diakses langsing melalui file hanya menampilkan isi kode file PHP, tidak seperti html yang dapat diakses secara langsung.
Bahan yang dibutuhkan antara lain:
Server
Xampp
Text Editor
VSCode

Syntax Lanjutan
Variable
  Variable merupakan tempat untuk menyimpan data sementara, variable bisa berupa string, interger, float dan lainya. Variable pada PHP memiliki ketentuan sebagai berikut:
1. Penulisan diawali dengan karakter $.
2. Bersifat Case Sensitive.
3. Dapat mengisi variable dengan tanda sama dengan “=”.

<?php
$nama = "Rama Sullivan";
$umur = 20;
echo $nama;
echo "<br/>
echo $umur;
echo "<br/>
echo "hai, perkenalkan nama saya ".$nama." dan saya berumur ".$umur;
?>

Sebagai pembanding dalam bahasa pemrograman python:

nama = "Rama Sullivan";
umur = 20
print (nama)
print ("n")
print (umur)
print ("\n")
print ("hai, perkenalkan nama saya "+nama+" dan saya berumur "+str(umur))

Kondisi If-Else
  Kondisi If-Else berguna untuk melakukan pengecekan suatu kondisi logika, apabila logika tersebut tercapai maka akan melakukan sebuah aksi dan jika tidak tercapai juga melakukan aksi.

<?php 
$fossclass = "Sabtu";
$ngobar = "Kamis";
if($fossclass != "Sabtu" && $ngobar == "Kamis"){
	echo " Hari ini tidak Fossclass tapi Ngobar ";
}elseif($fossclass == "Sabtu" && $ngobar != "Kamis"){
	echo " Hari ini Fossclass tapi tidak Ngobar ";
}elseif($fossclass == "Sabtu" && $ngobar == "Kamis"){
	echo " Hari ini Fossclass dan Ngobar ";
}else{
	echo "Hari ini tidak Fossclass dan tidak Ngobar";
}
?>

Sebagai pembanding dalam bahasa pemograman python:

fossclass = "Sabtu";
ngobar = "Kamis"
if(fossclass != "Sabtu" and ngobar == "Kamis"):
	print (" Hari ini tidak Fossclass tapi Ngobar ")
elif(fossclass == "Sabtu" and ngobar != "Kamis"):
	print (" Hari ini Fossclass tapi tidak Ngobar ")
elif(fossclass == "Sabtu" and ngobar == "Kamis"):
	print (" Hari ini Fossclass dan Ngobar ")
else:
	print ("Hari ini tidak Fossclass dan tidak Ngobar");

Fungsi
  Fungsi berguna untuk membungkus suatu blok kode program agar lebih efisien dalam pemanggilannya, dan dalam python adalah def.

<?php 
function tampilkan_nama(){
echo "Nama saya Rama Sullivan";
}

tampilkan_nama();
?>

Looping
  Loping digunakan intuk melakukan perulangan hingga suatu kondisi terpenuhi.

While Looping

<?php 
$x = 1; 
 
while($x <= 10) {
    echo "Angka $x 
";
    $x++;
} 
?>

For Looping

<?php 
for($x=1;$x<=10;$x++){
	echo $x;}
?>

Array
  Array merupakan suatu variable yang mampu menyimpan banyak data. Array biasa

<?php 
//membuat array yang berisi nama buah-buahan
$buah = array('semangka','jeruk','apel','anggur');
//menampilkan data array dengan nomor urut 2
echo $buah[2];
?>

Array dengan nama

<?php 
//penamaan isi array
$buah['semangka'] = "isinya merah";
$buah['jeruk'] = "rasanya manis";
$buah['apel'] = "warnanya merah";
$buah['anggur'] = "harganya mahal";

// menampilkan isi array yang bernama jeruk
echo $buah['jeruk'];
?>

<?php 
//penamaan isi array variabel buah
$buah = array(
	'semangka' => "isinya merah",
	'jeruk' => "rasanya manis",
	'apel' => "warnanya merah",
	'anggur' => "harganya mahal"
);

// menampilkan isi array yang bernama jeruk
echo $buah['jeruk'];
?>

Sebagai pembanding dalam bahasa pemograman python:

buah = ["anggur", "semangka", "jeruk"]
